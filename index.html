ApiBootstraper

ApiBootstraper is an Rest Api bootstrap that include an api server, api documentation and clients.

The purpose is to give an API bootstrap as well as his clients and so to facilitate the implementation of an API.
This bootstrap contains a system of authentification and bases itself on the example of a todo list.
